    -1️⃣ What is the difference between var, let, and const?

They are used to declare variables,but they behave differently.
var is the old way.It can be redeclared and updated.let is block scoped and can be updated but not redeclared in the same scope.const is also block-scoped but its value cannot be reassigned after declaration.Developers prefer let and const



    - 2️⃣ What is the spread operator (...)?

    It used to expand elements of an array or object.It is mainly used to copy or merge arrays and objects.It helps write cleaner and shorter code.



    - 3️⃣ What is the difference between map(), filter(), and forEach()?

  They are used with arrays. map() is creates a new array by modifying each element. filter() is creates a new array based on a condition. forEach()  is simply loops through the array and does not return a new array.
